# SORBA GitHub Organization

## What is SORBA?

SORBA is an AI automation platform designed to empower enterprises by improving asset reliability, optimizing processes, and accelerating sustainability through democratized, end-to-end Auto-ML solutions. It simplifies operations and doesn't require any coding experience or specialized data scientists. Therefore, it allows companies to focus on strategic business goals rather than spending hours deploying traditional machine learning models.

### Advantages

SORBA improves overall equipment effectiveness and product quality reducing energy consumption, preventing unplanned downtime, and reducing CO2 emissions. It enhances production lines to increase yield, and provides real-time recommendations tailored to control processes or assets.

For small and medium-sized enterprises, SORBA acts as a digital companion that not only identifies machine problems but uncovers the root cause. It learns in continuous operation, and becomes smarter with time ensuring businesses run smoothly and efficiently.

### Features

- **Easy Setup**—Web-based configuration. No dedicated hardware/software key required, only username & password.
- **Firewall Friendly**—Restful API web services allow for secure communications through corporate firewalls & cellular networks.
- **Integrated Communications**—WiFi, Ethernet, Cellular connectivity models available.
- **Secure Remote Access**—Access to remote IP devices such as PLC, Drives, Cameras, and route between networks.
- **Highest Level Security**—SSL Data encryption, OAuth 0-Legged session-based authentication, randomly generated software key.
- **Local Storage**—Event and error log on local storage SSD. Optional data collection and storage available with adjustable store and forward, push data throttle rate, and alarm notification.
- **AI Model Creation & Training**—Create and train machine learning models in a few simple steps using an intuitive interface, without requiring advanced AI knowledge.
